# X-Piratez Tech Tree Viewer

Инструмент для визуализации дерева технологий мода **X-Piratez** (для OpenXcom).

## Структура проекта

Проект разбит на модули для удобства разработки и поддержки:

```
├── config.py           # Конфигурация: пути, настройки
├── parser.py           # Парсинг файлов мода (.rul, .yml)
├── html_generator.py   # Генерация HTML с визуализацией
├── main.py             # Точка входа, объединяет все модули
├── tech_tree.py        # Оригинальный скрипт (для справки)
└── README.md           # Этот файл
```

## Как использовать

### Быстрый старт

1. Скопируйте эти 4 файла в папку с модом X-Piratez:
   - `config.py`
   - `parser.py`
   - `html_generator.py`
   - `main.py`

2. Запустите из корневой папки мода (где лежит папка `user/`):
   ```bash
   python main.py
   ```

3. Откройте созданный файл `pirate_viewer_v35.html` в браузере.

### Режим тестирования

Можно протестировать работу без файлов мода:
```bash
python main.py --test
```
Создастся файл `test_viewer.html` с демонстрационными данными.

### Аргументы командной строки

```bash
python main.py --help

--mod-path PATH    Путь к папке мода (если запускаете не из неё)
--test             Запуск с тестовыми данными
```

## Описание модулей

### `config.py`
- Пути к файлам мода (`Piratez.rul`, языковые файлы)
- Настройки по умолчанию (глубина графа, стартовая технология)
- **Модифицируйте этот файл**, если нужно изменить пути или настройки.

### `parser.py`
- Функции парсинга `.rul` файла (технологии, зависимости, разблокировки)
- Функции парсинга `.yml` языковых файлов (названия, описания UFOPEDIA)
- Очистка текста от служебных тегов
- **Модифицируйте**, если изменится формат файлов мода.

### `html_generator.py`
- HTML-шаблон с JavaScript для визуализации (библиотека vis-network)
- Логика генерации итогового файла
- Интерфейс на двух языках (EN/RU)
- **Модифицируйте**, если нужно изменить внешний вид или поведение графа.

### `main.py`
- Проверка наличия файлов
- Загрузка данных через `parser`
- Генерация HTML через `html_generator`
- Вывод статистики и результатов
- **Обычно не требует изменений**.

## Требования

- Python 3.6+
- Никаких внешних зависимостей (только стандартная библиотека)
- Браузер с поддержкой JavaScript для просмотра результата

## Особенности

- Поддержка английского и русского языков
- Поиск по технологиям (по названию и ID)
- Навигация по дереву (вперёд/назад)
- Настройка глубины отображения
- Ограничение на количество связей (для избежания перегрузки)
- Клик по узлу — показать детали
- Двойной клик — перейти к технологии
